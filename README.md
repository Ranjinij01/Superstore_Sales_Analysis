# Superstore Sales & Profit Analysis

## Objective
Analyze sales performance, profitability, and customer segments using SQL, Python, and Power BI.

## Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI

## Key Insights
- Identified top-performing categories and regions
- Found loss-making products impacting profitability
- Analyzed monthly sales trends for business decisions

## Dashboard
Interactive Power BI dashboard with KPIs and filters.
